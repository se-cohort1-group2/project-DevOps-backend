
### SQL database

access the database at [http://localhost:8080/h2-console](http://localhost:8080/h2-console).

|Key|Value|
|-|-|
|Driver Class|org.h2.Driver|
|JDBC URL|jdbc:h2:mem:m3db|
|User Name|sa|
|Password|password|
